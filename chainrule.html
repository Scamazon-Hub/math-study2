The Chain Rule - Function of a Function Differentiation

The Chain Rule

Mastering Composite Function Differentiation

The chain rule is one of the most powerful tools in calculus, allowing us to differentiate complex functions built from simpler components. It's essential for engineering applications involving rates of change in interconnected systems.

1. Understanding Cascade Relationships

What is a Cascade Relationship?

In engineering systems, variables are often linked indirectly through intermediate relationships. Understanding these cascading dependencies is crucial for analyzing complex systems.

Mathematical Cascade

When a variable \(y\) depends on \(z\), which in turn depends on \(x\), we have a chain relationship:

Engineering Applications

Engineering Field Cascade Example Mathematical Representation
Thermodynamics Fuel consumption → Heat input → Temperature \(T = f(H), H = f(C)\)
Control Systems Input voltage → Motor speed → Position \(θ = f(ω), ω = f(V)\)
Fluid Dynamics Pressure → Flow rate → Turbine power \(P = f(Q), Q = f(p)\)

Example: Thermal System Analysis

A heating system where temperature rise depends on heat input, which depends on fuel consumption rate.

Engineering Insight: The chain rule allows us to find how temperature changes with fuel consumption: \(\frac{dT}{dC} = \frac{dT}{dH} \times \frac{dH}{dC}\)

2. The Chain Rule Formula

Fundamental Principle

The chain rule provides a method for differentiating composite functions by multiplying the derivatives of the individual functions in the chain.

Basic Chain Rule

$$ \frac{dy}{dx} = \frac{dy}{dz} \times \frac{dz}{dx} $$

Extended Chain Rule

For longer chains with more variables:

$$ \frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dv} \times \frac{dv}{dx} $$

Leibniz Notation Insight

Visualizing the Cancellation

While not mathematically rigorous, the "cancellation" of intermediate differentials provides an intuitive way to remember and apply the chain rule: $$ \frac{dy}{dx} = \frac{dy}{\cancel{dz}} \times \frac{\cancel{dz}}{dx} $$

3. Engineering Applications

Practical Implementation

The chain rule finds extensive applications across various engineering disciplines for analyzing rates of change in complex systems.

Mechanical Engineering: Kinematics

Example: Robot Arm Motion

A robotic arm has joint angle θ that depends on motor speed ω, which depends on input voltage V. Find how joint position changes with voltage.

Given:
Position: \(x = 2\sinθ\)
Angular velocity: \(θ = 0.5ωt\)
Motor response: \(ω = 3V^2\)

Solution using chain rule:
\(\frac{dx}{dV} = \frac{dx}{dθ} \times \frac{dθ}{dω} \times \frac{dω}{dV}\)
\(\frac{dx}{dθ} = 2\cosθ\)
\(\frac{dθ}{dω} = 0.5t\)
\(\frac{dω}{dV} = 6V\)
\(\frac{dx}{dV} = 2\cosθ \times 0.5t \times 6V = 6Vt\cosθ\)

Electrical Engineering: Power Systems

Example: Power Loss in Transmission

Power loss in a transmission line depends on current, which depends on load impedance, which depends on temperature.

Given:
Power loss: \(P = I^2R\)
Current: \(I = \frac{V}{Z}\)
Impedance: \(Z = Z_0(1 + αT)\)

Find rate of power loss change with temperature:
\(\frac{dP}{dT} = \frac{dP}{dI} \times \frac{dI}{dZ} \times \frac{dZ}{dT}\)
\(\frac{dP}{dI} = 2IR\)
\(\frac{dI}{dZ} = -\frac{V}{Z^2}\)
\(\frac{dZ}{dT} = Z_0α\)
\(\frac{dP}{dT} = 2IR \times (-\frac{V}{Z^2}) \times Z_0α = -\frac{2IVRZ_0α}{Z^2}\)

4. Advanced Techniques

Multiple Chain Rules

Handling Complex Dependencies

Some functions require applying the chain rule multiple times or in combination with other differentiation rules.

Example: Triple Composition

Differentiate \(y = \sin(\ln(x^2 + 1))\)

Step 1: Identify the chain
Let \(u = x^2 + 1\), \(v = \ln u\), \(y = \sin v\)

Step 2: Apply chain rule
\(\frac{dy}{dx} = \frac{dy}{dv} \times \frac{dv}{du} \times \frac{du}{dx}\)
\(\frac{dy}{dv} = \cos v\)
\(\frac{dv}{du} = \frac{1}{u}\)
\(\frac{du}{dx} = 2x\)
\(\frac{dy}{dx} = \cos v \times \frac{1}{u} \times 2x = \frac{2x\cos(\ln(x^2 + 1))}{x^2 + 1}\)

Chain Rule with Other Rules

Combination When to Use Example
Chain + Product Rule Product of composite functions \(y = (x^2+1)^3 \cdot \sin(2x)\)
Chain + Quotient Rule Quotient of composite functions \(y = \frac{\cos(3x)}{(2x-1)^2}\)
Multiple Chain Rules Nested composite functions \(y = \ln(\sin(e^x))\)

5. Comprehensive Worked Examples

Example 1: Basic Chain Rule

Given \(y = (3x^2 + 2x)^4\), find \(\frac{dy}{dx}\)

Method 1: Direct Substitution
Expand: \(y = (3x^2 + 2x)^4\) (complicated expansion)
Differentiate term-by-term (tedious)

Method 2: Chain Rule (Recommended)
Let \(u = 3x^2 + 2x\), then \(y = u^4\)
\(\frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx} = 4u^3 \times (6x + 2)\)
Substitute back: \(\frac{dy}{dx} = 4(3x^2 + 2x)^3(6x + 2)\)

Example 2: Engineering Application

The deflection δ of a beam under load is given by \(\delta = \frac{PL^3}{3EI}\), where I depends on beam dimensions. If I = k·b·h³, find how deflection changes with beam height h.

Solution:
\(\frac{d\delta}{dh} = \frac{d\delta}{dI} \times \frac{dI}{dh}\)
\(\frac{d\delta}{dI} = -\frac{PL^3}{3EI^2}\)
\(\frac{dI}{dh} = 3k·b·h^2\)
\(\frac{d\delta}{dh} = -\frac{PL^3}{3EI^2} \times 3k·b·h^2 = -\frac{PL^3k·b·h^2}{EI^2}\)

Engineering Interpretation: Deflection decreases as beam height increases (negative derivative).

Test Your Understanding

Try these problems to reinforce your chain rule skills. Solutions are provided for self-checking.

  2. Find \(\frac{dy}{dx}\) for \(y = \sqrt{4x^2 + 3x}\)

Engineering Problems

  1. The period T of a pendulum is \(T = 2\pi\sqrt{\frac{L}{g}}\). Find how T changes with length L.
  2. In an RC circuit, voltage across capacitor is \(V_c = V_0(1 - e^{-t/RC})\). Find rate of voltage change with time.
  3. The stress in a pressure vessel is \(\sigma = \frac{pr}{t}\), where t depends on temperature. Find \(\frac{d\sigma}{dT}\).
Engineering Insight: The chain rule transforms complex engineering problems into manageable steps. Always identify the intermediate variables and their relationships before applying the rule.

7. Summary and Key Points

Essential Takeaways

Mastery of the chain rule is fundamental for engineering mathematics and practical problem-solving.
Concept Key Formula Engineering Application
Basic Chain Rule \(\frac{dy}{dx} = \frac{dy}{dz} \cdot \frac{dz}{dx}\) Single intermediate variable systems
Extended Chain Rule \(\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dv} \cdot \frac{dv}{dx}\) Multiple intermediate variables
Combined Rules Chain + Product/Quotient rules Complex engineering systems
Common Pitfall: Forgetting to multiply by the derivative of the inner function is the most common error. Always differentiate from the outside inward.
