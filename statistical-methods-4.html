Linear Correlation and Regression

Analyzing Relationships and Building Predictive Models

Correlation and regression analysis provide powerful tools for understanding relationships between variables and developing predictive models essential for engineering design, optimization, and control.

Introduction to Linear Correlation and Regression

From Description to Prediction

While descriptive statistics help us understand individual variables, correlation and regression analysis reveal how variables relate to each other, enabling prediction and optimization in engineering systems.

Engineering Applications

Correlation and regression are fundamental to:

  • Process Optimization: Relating process parameters to product quality
  • Design Engineering: Predicting performance from design variables
  • Quality Control: Identifying relationships between process variables
  • Reliability Engineering: Modeling failure mechanisms
  • Experimental Design: Analyzing factor effects and interactions

Linear Correlation

Measuring Relationship Strength

Correlation analysis quantifies the strength and direction of linear relationships between variables, providing the foundation for understanding causal relationships in engineering systems.

Pearson Correlation Coefficient

Pearson's r Formula:

\[ r = \frac{\sum{(x_i - \bar{x})(y_i - \bar{y})}}{\sqrt{\sum{(x_i - \bar{x})^2}\sum{(y_i - \bar{y})^2}}} \]

Simplified Computational Formula:

\[ r = \frac{n\sum{xy} - \sum{x}\sum{y}}{\sqrt{[n\sum{x^2} - (\sum{x})^2][n\sum{y^2} - (\sum{y})^2]}} \]

Correlation Calculation Example

Calculate the correlation between engine speed (RPM) and fuel consumption (L/h):

Speed (x) Fuel (y) x² y² xy
1500 8.2 2,250,000 67.24 12,300
2000 10.5 4,000,000 110.25 21,000
2500 13.1 6,250,000 171.61 32,750
3000 16.3 9,000,000 265.69 48,900
3500 20.1 12,250,000 404.01 70,350
Σ 12500 68.2 33,750,000 1,018.8

Calculation:

n = 5, Σx = 12500, Σy = 68.2, Σx² = 33,750,000, Σy² = 1018.8, Σxy = 185,300

\[ r = \frac{5(185300) - (12500)(68.2)}{\sqrt{[5(33750000) - (12500)^2][5(1018.8) - (68.2)^2]}} \]

\[ r = \frac{926500 - 852500}{\sqrt{[168750000 - 156250000][5094 - 4651.24]}} \]

\[ r = \frac{74000}{\sqrt{[12500000][442.76]}} = \frac{74000}{\sqrt{5534500000}} = \frac{74000}{74393} = 0.994 \]

Interpretation: r = 0.994 indicates a very strong positive linear relationship between engine speed and fuel consumption.

Interpreting Correlation Coefficients

r Value Strength Direction Engineering Implication
0.9 to 1.0 Very strong Positive Strong predictive relationship
0.7 to 0.9 Strong Positive Good for prediction
0.5 to 0.7 Moderate Positive Useful relationship
0.3 to 0.5 Weak Positive Limited predictive value
0.0 to 0.3 Very weak None No practical relationship

Correlation vs. Causation: High correlation does not imply causation. Always consider engineering principles and potential confounding variables when interpreting correlation results.

Regression

Building Predictive Models

Regression analysis develops mathematical models that predict one variable from another, enabling engineers to optimize processes, predict performance, and understand system behavior.

Simple Linear Regression Model

Regression Equation:

\[ y = a + bx \]

where:

• y = dependent variable (response)

• x = independent variable (predictor)

• a = y-intercept (constant term)

• b = slope (regression coefficient)

Least Squares Estimation

Slope Formula:

\[ b = \frac{n\sum{xy} - \sum{x}\sum{y}}{n\sum{x^2} - (\sum{x})^2} \]

Intercept Formula:

\[ a = \bar{y} - b\bar{x} \]

Regression Model Development

Using the engine speed vs. fuel consumption data from the correlation example:

Step 1: Calculate slope (b)

\[ b = \frac{5(185300) - (12500)(68.2)}{5(33750000) - (12500)^2} = \frac{926500 - 852500}{168750000 - 156250000} = \frac{74000}{12500000} = 0.00592 \]

Step 2: Calculate means

\[ \bar{x} = \frac{12500}{5} = 2500, \quad \bar{y} = \frac{68.2}{5} = 13.64 \]

Step 3: Calculate intercept (a)

\[ a = 13.64 - 0.00592(2500) = 13.64 - 14.8 = -1.16 \]

Step 4: Regression equation

\[ \hat{y} = -1.16 + 0.00592x \]

Interpretation: For every 1 RPM increase in engine speed, fuel consumption increases by 0.00592 L/h.

Regression Diagnostics and Validation

Model Assessment and Prediction

Using the regression equation: \( \hat{y} = -1.16 + 0.00592x \)

Prediction at x = 2750 RPM:

\[ \hat{y} = -1.16 + 0.00592(2750) = -1.16 + 16.28 = 15.12 \text{ L/h} \]

Goodness of Fit - Coefficient of Determination (R²):

\[ R^2 = r^2 = (0.994)^2 = 0.988 \]

98.8% of the variation in fuel consumption is explained by engine speed.

Residual Analysis:

Residual = Observed - Predicted

Small, randomly distributed residuals indicate a good model fit.

Multiple Linear Regression

Multiple Regression Model:

\[ y = a + b_1x_1 + b_2x_2 + \cdots + b_kx_k \]

where multiple predictor variables are used to improve prediction accuracy.

Engineering Optimization Example

A chemical process has yield (y) dependent on temperature (x₁) and pressure (x₂):

\[ \hat{y} = 25.3 + 0.85x_1 + 1.2x_2 \]

Interpretation:

• Holding pressure constant, each 1°C temperature increase improves yield by 0.85%

• Holding temperature constant, each 1 bar pressure increase improves yield by 1.2%

• The model can be used to optimize process conditions for maximum yield

Regression Assumptions and Limitations

Key Assumptions for Valid Regression:

  • Linearity: Relationship between variables is linear
  • Independence: Observations are independent
  • Homoscedasticity: Constant variance of errors
  • Normality: Errors are normally distributed
  • No Multicollinearity: Predictor variables aren't highly correlated

Summary of Linear Correlation and Regression

Powerful Tools for Engineering Analysis

Correlation and regression provide the mathematical foundation for understanding relationships, developing predictive models, and optimizing engineering systems.

Key Concepts and Formulas

  • Correlation Coefficient (r): Measures strength and direction of linear relationships (-1 to +1)
  • Regression Equation: y = a + bx predicts dependent variable from independent variable
  • Least Squares Method: Minimizes sum of squared residuals to find best-fit line
  • Coefficient of Determination (R²): Proportion of variance explained by the model (0 to 1)
  • Residual Analysis: Validates model assumptions and identifies problems
