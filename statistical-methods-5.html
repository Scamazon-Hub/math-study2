Probability

The Mathematics of Uncertainty

Probability theory provides the mathematical foundation for dealing with uncertainty in engineering systems, enabling rational decision-making under conditions of incomplete information.

Introduction to Probability

Quantifying Uncertainty in Engineering

From reliability analysis to risk assessment, probability concepts allow engineers to make informed decisions despite the inherent uncertainties in materials, processes, and operating conditions.

Engineering Applications of Probability

Probability theory supports critical engineering functions:

  • Reliability Engineering: Predicting system failure probabilities
  • Risk Assessment: Quantifying and managing technical risks
  • Quality Control: Determining sampling plans and acceptance criteria
  • Decision Analysis: Choosing between alternative designs or strategies
  • Safety Engineering: Assessing probability of hazardous events

The Classical Definition of Probability

Foundational Concepts

The classical definition of probability provides the theoretical foundation for calculating likelihoods in situations where all outcomes are equally likely.

Basic Probability Concepts

Sample Space (S): The set of all possible outcomes

Event (E): A subset of the sample space

Classical Probability Definition:

\[ P(E) = \frac{\text{Number of favorable outcomes}}{\text{Total number of possible outcomes}} \]

Engineering Component Selection

A batch of 100 electronic components contains 5 defective units. If one component is selected at random:

Sample Space: 100 components

Event E: Selecting a defective component

Probability Calculation:

\[ P(\text{defective}) = \frac{5}{100} = 0.05 \]

There is a 5% probability of selecting a defective component.

Probability Axioms

Axiom 1: \( 0 \leq P(E) \leq 1 \) for any event E

Axiom 2: \( P(S) = 1 \)

Axiom 3: For mutually exclusive events E₁, E₂, ..., Eₙ:

\[ P(E_1 \cup E_2 \cup \cdots \cup E_n) = P(E_1) + P(E_2) + \cdots + P(E_n) \]

Addition of Probabilities

Combining Events

Addition rules allow engineers to calculate probabilities of combined events, essential for analyzing complex systems with multiple failure modes or success criteria.

General Addition Rule

For any two events A and B:

\[ P(A \cup B) = P(A) + P(B) - P(A \cap B) \]

Quality Control Application

A manufacturing process produces components that may have dimensional errors (D) or surface defects (S). Historical data shows:

P(D) = 0.08, P(S) = 0.05, P(D ∩ S) = 0.02

Question: What is the probability a component has at least one type of defect?

Solution:

\[ P(D \cup S) = P(D) + P(S) - P(D \cap S) = 0.08 + 0.05 - 0.02 = 0.11 \]

11% of components have at least one type of defect.

Question: What is the probability a component has only one type of defect?

\[ P(\text{only one defect}) = P(D \cup S) - P(D \cap S) = 0.11 - 0.02 = 0.09 \]

Alternatively: \( P(D \cap S') + P(D' \cap S) = (0.08 - 0.02) + (0.05 - 0.02) = 0.06 + 0.03 = 0.09 \)

Mutually Exclusive Events

If events A and B are mutually exclusive (cannot occur together):

\[ P(A \cap B) = 0 \]

\[ P(A \cup B) = P(A) + P(B) \]

Reliability Analysis

A system has three independent failure modes with probabilities:

P(mode 1) = 0.02, P(mode 2) = 0.015, P(mode 3) = 0.025

Question: What is the probability the system fails?

Solution: Since failure modes are mutually exclusive:

\[ P(\text{failure}) = 0.02 + 0.015 + 0.025 = 0.06 \]

The system has a 6% probability of failure.

Conditional Probability

Updating Beliefs with New Information

Conditional probability allows engineers to update probability assessments as new information becomes available, crucial for diagnostic systems and real-time decision making.

Conditional Probability Definition

Conditional Probability Formula:

\[ P(A|B) = \frac{P(A \cap B)}{P(B)} \quad \text{for } P(B) > 0 \]

Manufacturing Process Diagnosis

A production line has two machines. Machine A produces 60% of output with 3% defect rate. Machine B produces 40% of output with 5% defect rate.

Given: A component is found to be defective. What is the probability it came from Machine A?

Solution using Conditional Probability:

P(A) = 0.60, P(B) = 0.40, P(D|A) = 0.03, P(D|B) = 0.05

\[ P(D) = P(D|A)P(A) + P(D|B)P(B) = (0.03)(0.60) + (0.05)(0.40) = 0.018 + 0.02 = 0.038 \]

\[ P(A|D) = \frac{P(D|A)P(A)}{P(D)} = \frac{0.018}{0.038} = 0.474 \]

Given a defective component, there is a 47.4% probability it came from Machine A.

Independent Events

Events A and B are independent if:

\[ P(A|B) = P(A) \quad \text{or} \quad P(B|A) = P(B) \]

Equivalently:

\[ P(A \cap B) = P(A)P(B) \]

Multiplication of Probabilities

Sequential Events and System Reliability

Multiplication rules enable engineers to calculate probabilities of event sequences and analyze complex systems with multiple components in series or parallel configurations.

General Multiplication Rule

For any two events A and B:

\[ P(A \cap B) = P(A|B)P(B) = P(B|A)P(A) \]

Quality Control Sampling

A batch of 20 components contains 3 defectives. Two components are selected without replacement.

Question: What is the probability both are defective?

Solution:

Let D₁ = first is defective, D₂ = second is defective

\[ P(D_1) = \frac{3}{20} = 0.15 \]

\[ P(D_2|D_1) = \frac{2}{19} \approx 0.1053 \]

\[ P(D_1 \cap D_2) = P(D_1)P(D_2|D_1) = 0.15 \times 0.1053 = 0.0158 \]

There is approximately a 1.58% probability that both selected components are defective.

System Reliability Analysis

Series System Reliability

A system with three components in series fails if any component fails. Component reliabilities: R₁ = 0.95, R₂ = 0.98, R₃ = 0.92

System Reliability:

\[ R_{system} = R_1 \times R_2 \times R_3 = 0.95 \times 0.98 \times 0.92 = 0.856 \]

Parallel System Reliability

A system with two components in parallel fails only if both components fail. Component reliabilities: R₁ = 0.95, R₂ = 0.90

System Reliability:

\[ R_{system} = 1 - (1 - R_1)(1 - R_2) = 1 - (0.05)(0.10) = 1 - 0.005 = 0.995 \]

Bayes' Theorem

Inverse Probability and Diagnostic Systems

Bayes' Theorem provides a systematic method for updating probabilities based on new evidence, forming the foundation for diagnostic systems, pattern recognition, and decision analysis.

Bayes' Theorem Formula

Bayes' Theorem:

\[ P(A|B) = \frac{P(B|A)P(A)}{P(B)} \]

Extended form using total probability:

\[ P(A_i|B) = \frac{P(B|A_i)P(A_i)}{\sum_{j=1}^n P(B|A_j)P(A_j)} \]

Medical Diagnostic System

A diagnostic test for a rare engineering material defect:

• Prevalence of defect: P(D) = 0.01

• Test sensitivity: P(T|D) = 0.95 (probability test positive given defect)

• Test specificity: P(T'|D') = 0.90 (probability test negative given no defect)

Question: If the test is positive, what is the probability the material actually has the defect?

Solution using Bayes' Theorem:

\[ P(D|T) = \frac{P(T|D)P(D)}{P(T|D)P(D) + P(T|D')P(D')} \]

\[ P(T|D') = 1 - P(T'|D') = 1 - 0.90 = 0.10 \]

\[ P(D|T) = \frac{0.95 \times 0.01}{0.95 \times 0.01 + 0.10 \times 0.99} = \frac{0.0095}{0.0095 + 0.099} = \frac{0.0095}{0.1085} \approx 0.0876 \]

Even with a positive test result, there is only an 8.76% probability the material actually has the defect.

Summary of Probability

Complete Framework for Uncertainty Analysis

Probability theory provides engineers with a comprehensive toolkit for quantifying, analyzing, and managing uncertainty across all aspects of engineering practice.

Probability Rules Summary

  • Classical Probability: P(E) = favorable outcomes / total outcomes
  • Addition Rule: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
  • Conditional Probability: P(A|B) = P(A ∩ B) / P(B)
  • Multiplication Rule: P(A ∩ B) = P(A|B)P(B) = P(B|A)P(A)
  • Bayes' Theorem: P(A|B) = P(B|A)P(A) / P(B)
  • Independence: P(A ∩ B) = P(A)P(B)

Comprehensive Formula Collection

A complete reference of probability formulas and relationships for engineering applications.

Basic Probability Formulas

Complement Rule: \( P(A') = 1 - P(A) \)

Union of Events: \( P(A \cup B) = P(A) + P(B) - P(A \cap B) \)

Mutually Exclusive Events: \( P(A \cup B) = P(A) + P(B) \)

Conditional Probability: \( P(A|B) = \frac{P(A \cap B)}{P(B)} \)

Multiplication Rule: \( P(A \cap B) = P(A|B)P(B) = P(B|A)P(A) \)

Independent Events: \( P(A \cap B) = P(A)P(B) \)

Total Probability: \( P(B) = \sum P(B|A_i)P(A_i) \)

Bayes' Theorem: \( P(A_j|B) = \frac{P(B|A_j)P(A_j)}{\sum P(B|A_i)P(A_i)} \)

System Reliability Formulas

Series System: \( R_{system} = \prod_{i=1}^n R_i \)

Parallel System: \( R_{system} = 1 - \prod_{i=1}^n (1 - R_i) \)

k-out-of-n System: \( R_{system} = \sum_{i=k}^n \binom{n}{i} R^i (1-R)^{n-i} \)
