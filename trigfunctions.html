Trigonometric Functions Integration - Sine and Cosine

Integration of Trigonometric Functions

Extending the Inverse Approach to Trigonometry

The inverse differentiation method provides a powerful technique for deriving integration rules for trigonometric functions, particularly sine and cosine, which are fundamental in engineering applications involving periodic phenomena.

1. Basic Trigonometric Integration Rules

Reversing Trigonometric Differentiation

By applying the inverse approach to the derivatives of sine and cosine functions, we can derive their corresponding integration formulas.

Derivation from First Principles

Sine Function Integration

Step 1: Recall the derivative
\[ \frac{d}{dx}[\sin x] = \cos x \]

Step 2: Apply the inverse principle
\[ \int \cos x \, dx = \sin x + c \]

Step 3: Verification
\[ \frac{d}{dx}[\sin x + c] = \cos x \quad \checkmark \]

Cosine Function Integration

Step 1: Recall the derivative
\[ \frac{d}{dx}[\cos x] = -\sin x \]

Step 2: Apply the inverse principle
\[ \int (-\sin x) \, dx = \cos x + c \]

Step 3: Solve for ∫sin x dx
\[ -\int \sin x \, dx = \cos x + c \]
\[ \int \sin x \, dx = -\cos x + c \]

Step 4: Verification
\[ \frac{d}{dx}[-\cos x + c] = -(-\sin x) = \sin x \quad \checkmark \]

Summary of Basic Rules

\[ \int \sin x \, dx = -\cos x + c \]

\[ \int \cos x \, dx = \sin x + c \]

2. Generalized Trigonometric Integration

Extending to Functions of Linear Forms

Many engineering applications involve trigonometric functions with arguments of the form (ax + b), requiring generalized integration rules.

Derivation Using Chain Rule

Cosine Function with Linear Argument

Step 1: Differentiate sin(ax + b)
Using the chain rule:
\[ \frac{d}{dx}[\sin(ax + b)] = \cos(ax + b) \cdot a = a\cos(ax + b) \]

Step 2: Apply the inverse principle
\[ \int a\cos(ax + b) \, dx = \sin(ax + b) + c \]

Step 3: Solve for ∫cos(ax + b) dx
\[ \int \cos(ax + b) \, dx = \frac{1}{a} \sin(ax + b) + c \]

Sine Function with Linear Argument

Step 1: Differentiate cos(ax + b)
Using the chain rule:
\[ \frac{d}{dx}[\cos(ax + b)] = -\sin(ax + b) \cdot a = -a\sin(ax + b) \]

Step 2: Apply the inverse principle
\[ \int (-a\sin(ax + b)) \, dx = \cos(ax + b) + c \]

Step 3: Solve for ∫sin(ax + b) dx
\[ -\int a\sin(ax + b) \, dx = \cos(ax + b) + c \]
\[ \int \sin(ax + b) \, dx = -\frac{1}{a} \cos(ax + b) + c \]

Complete Set of Generalized Rules

\[ \int \sin(ax + b) \, dx = -\frac{1}{a} \cos(ax + b) + c \]

\[ \int \cos(ax + b) \, dx = \frac{1}{a} \sin(ax + b) + c \]

3. Worked Examples and Applications

Practical Implementation

These examples demonstrate the application of trigonometric integration rules to various engineering scenarios, including those with numerical coefficients and definite integrals.

Example 1: Basic Trigonometric Integration with Coefficient

Integrate \( f(x) = 3 \sin(0.2x + 0.5) \).

Step 1: Extract the constant
\[ I = \int 3 \sin(0.2x + 0.5) \, dx = 3 \int \sin(0.2x + 0.5) \, dx \]

Step 2: Identify parameters
\( a = 0.2 \), \( b = 0.5 \)

Step 3: Apply the integration rule
\[ I = 3 \left[ -\frac{1}{0.2} \cos(0.2x + 0.5) \right] + c = 3 \left[ -5 \cos(0.2x + 0.5) \right] + c \]

Step 4: Simplify
\[ I = -15 \cos(0.2x + 0.5) + c \]

Step 5: Verification
\[ \frac{d}{dx}[-15 \cos(0.2x + 0.5) + c] = -15 \cdot (-\sin(0.2x + 0.5) \cdot 0.2) = 3 \sin(0.2x + 0.5) \quad \checkmark \]

Engineering Context: This type of function models oscillatory systems with specific amplitude and phase characteristics.

Example 2: Definite Integral with Angle Parameter

Evaluate \( \int_{1.2}^{1.5} 5 \cos(2\theta - 1) \, d\theta \).

Step 1: Extract the constant
\[ I = \int_{1.2}^{1.5} 5 \cos(2\theta - 1) \, d\theta = 5 \int_{1.2}^{1.5} \cos(2\theta - 1) \, d\theta \]

Step 2: Identify parameters
\( a = 2 \), \( b = -1 \)

Step 3: Apply the integration rule
\[ I = 5 \left[ \frac{1}{2} \sin(2\theta - 1) \right]_{1.2}^{1.5} = 2.5 \left[ \sin(2\theta - 1) \right]_{1.2}^{1.5} \]

Step 4: Evaluate at upper limit (θ = 1.5)
\[ \sin(2(1.5) - 1) = \sin(3 - 1) = \sin(2) \approx 0.9093 \]

Step 5: Evaluate at lower limit (θ = 1.2)
\[ \sin(2(1.2) - 1) = \sin(2.4 - 1) = \sin(1.4) \approx 0.9854 \]

Step 6: Compute the difference
\[ I = 2.5 (0.9093 - 0.9854) = 2.5 (-0.0761) = -0.19025 \]

Step 7: Final result
\[ I \approx -0.190 \quad \text{(to 3 significant figures)} \]

Engineering Interpretation: This definite integral represents the net area under a cosine wave between specific angular values, useful in signal processing and vibration analysis.

4. Additional Examples and Engineering Applications

Expanding the Scope

Trigonometric integration finds applications in various engineering fields, including electrical engineering, mechanical vibrations, and wave propagation.

Example 3: Electrical Engineering Application

In an AC circuit, the current is given by \( i(t) = 10 \sin(100\pi t + \frac{\pi}{4}) \) amperes. Find the charge transferred between t = 0 and t = 0.01 seconds.

Step 1: Charge-current relationship
Charge \( Q = \int i \, dt \)

Step 2: Set up the integral
\[ Q = \int_{0}^{0.01} 10 \sin(100\pi t + \frac{\pi}{4}) \, dt \]

Step 3: Extract the constant
\[ Q = 10 \int_{0}^{0.01} \sin(100\pi t + \frac{\pi}{4}) \, dt \]

Step 4: Identify parameters
\( a = 100\pi \), \( b = \frac{\pi}{4} \)

Step 5: Apply the integration rule
\[ Q = 10 \left[ -\frac{1}{100\pi} \cos(100\pi t + \frac{\pi}{4}) \right]_{0}^{0.01} \]

Step 6: Simplify
\[ Q = -\frac{10}{100\pi} \left[ \cos(100\pi t + \frac{\pi}{4}) \right]_{0}^{0.01} = -\frac{1}{10\pi} \left[ \cos(100\pi t + \frac{\pi}{4}) \right]_{0}^{0.01} \]

Step 7: Evaluate at limits
At \( t = 0.01 \): \( \cos(100\pi(0.01) + \frac{\pi}{4}) = \cos(\pi + \frac{\pi}{4}) = \cos(\frac{5\pi}{4}) = -\frac{\sqrt{2}}{2} \)
At \( t = 0 \): \( \cos(0 + \frac{\pi}{4}) = \cos(\frac{\pi}{4}) = \frac{\sqrt{2}}{2} \)

Step 8: Compute the difference
\[ Q = -\frac{1}{10\pi} \left( -\frac{\sqrt{2}}{2} - \frac{\sqrt{2}}{2} \right) = -\frac{1}{10\pi} (-\sqrt{2}) = \frac{\sqrt{2}}{10\pi} \]

Step 9: Numerical result
\[ Q \approx \frac{1.4142}{31.4159} \approx 0.0450 \, \text{coulombs} \]

Example 4: Mechanical Vibration Analysis

The velocity of a vibrating mass is given by \( v(t) = 0.5 \cos(8t - \frac{\pi}{3}) \) m/s. Find the displacement function if the initial displacement is 0.1 meters.

Step 1: Displacement-velocity relationship
Displacement \( s(t) = \int v(t) \, dt \)

Step 2: Set up the integral
\[ s(t) = \int 0.5 \cos(8t - \frac{\pi}{3}) \, dt \]

Step 3: Extract the constant
\[ s(t) = 0.5 \int \cos(8t - \frac{\pi}{3}) \, dt \]

Step 4: Identify parameters
\( a = 8 \), \( b = -\frac{\pi}{3} \)

Step 5: Apply the integration rule
\[ s(t) = 0.5 \left[ \frac{1}{8} \sin(8t - \frac{\pi}{3}) \right] + c = 0.0625 \sin(8t - \frac{\pi}{3}) + c \]

Step 6: Apply initial condition
\( s(0) = 0.1 \): \( 0.1 = 0.0625 \sin(-\frac{\pi}{3}) + c \)
\( 0.1 = 0.0625 (-\frac{\sqrt{3}}{2}) + c \)
\( 0.1 = -0.0541 + c \)
\( c = 0.1541 \)

Step 7: Final displacement function
\[ s(t) = 0.0625 \sin(8t - \frac{\pi}{3}) + 0.1541 \]

5. Common Integration Patterns

Recognizing Standard Forms

Many trigonometric integration problems follow recognizable patterns. Identifying these patterns can significantly simplify the solution process.

Standard Trigonometric Integration Formulas

Function Integral Notes
\( \int \sin x \, dx \) \( -\cos x + c \) Basic sine integral
\( \int \cos x \, dx \) \( \sin x + c \) Basic cosine integral
\( \int \sin(ax + b) \, dx \) \( -\frac{1}{a} \cos(ax + b) + c \) Generalized sine integral
\( \int \cos(ax + b) \, dx \) \( \frac{1}{a} \sin(ax + b) + c \) Generalized cosine integral
\( \int k \cdot \sin(ax + b) \, dx \) \( -\frac{k}{a} \cos(ax + b) + c \) With constant coefficient
\( \int k \cdot \cos(ax + b) \, dx \) \( \frac{k}{a} \sin(ax + b) + c \) With constant coefficient

6. Advanced Techniques and Tips

Enhancing Problem-Solving Skills

Beyond the basic rules, several techniques can help solve more complex trigonometric integration problems efficiently.

Strategy for Trigonometric Integration

  1. Identify the form: Determine if the integrand matches a standard trigonometric pattern
  2. Extract constants: Factor out numerical coefficients before integrating
  3. Parameter identification: Clearly identify a and b in expressions of the form (ax + b)
  4. Verification: Always differentiate your result to verify correctness
  5. Unit consistency: Ensure mathematical operations maintain dimensional consistency in engineering applications

Common Pitfalls to Avoid:

  • Forgetting to divide by the coefficient 'a' in generalized forms
  • Mishandling negative signs, especially with sine integrals
  • Using degrees instead of radians in calculus operations
  • Omitting the constant of integration in indefinite integrals
  • Incorrectly applying limits in definite integrals
Engineering Application Insight: Trigonometric integration is fundamental in analyzing periodic systems. The ability to integrate sine and cosine functions efficiently is crucial for solving differential equations that model oscillatory behavior in electrical circuits, mechanical vibrations, and wave phenomena.
