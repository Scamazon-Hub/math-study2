Probability Density

From Frequency to Probability

Probability density functions extend the concept of frequency distributions to continuous random variables, forming the mathematical foundation for statistical inference.

Probability Density Functions (PDFs)

For a continuous random variable X with PDF f(x):

\[ P(a \leq X \leq b) = \int_a^b f(x) \, dx \]

Properties:

1. \( f(x) \geq 0 \) for all x

2. \( \int_{-\infty}^{\infty} f(x) \, dx = 1 \)

Uniform Distribution Example

A manufacturing process produces components with diameters uniformly distributed between 49.5 mm and 50.5 mm.

PDF: \( f(x) = \begin{cases} 1 & \text{for } 49.5 \leq x \leq 50.5 \\ 0 & \text{otherwise} \end{cases} \)

Probability Calculation: What's the probability a component has diameter between 49.8 mm and 50.2 mm?

\[ P(49.8 \leq X \leq 50.2) = \int_{49.8}^{50.2} 1 \, dx = 50.2 - 49.8 = 0.4 \]

40% of components fall within this tolerance range.

Relationship Between PDF and CDF

Cumulative Distribution Function (CDF):

\[ F(x) = P(X \leq x) = \int_{-\infty}^x f(t) \, dt \]

Relationship:

\[ f(x) = \frac{d}{dx} F(x) \]

Probabilities

Quantifying Uncertainty

Probability theory provides the mathematical framework for dealing with uncertainty in engineering systems, from reliability analysis to risk assessment.

Basic Probability Concepts

Sample Space (S): The set of all possible outcomes

Event (E): A subset of the sample space

Probability Axioms:

1. \( 0 \leq P(E) \leq 1 \)

2. \( P(S) = 1 \)

3. For mutually exclusive events: \( P(E_1 \cup E_2) = P(E_1) + P(E_2) \)

Engineering Reliability Example

A system has three critical components with individual reliabilities:

Component A: 0.95, Component B: 0.98, Component C: 0.92

Scenario 1: Series System (all must work)

\[ P(\text{system works}) = 0.95 \times 0.98 \times 0.92 = 0.856 \]

Scenario 2: Parallel System (at least one must work)

\[ P(\text{system fails}) = (1-0.95) \times (1-0.98) \times (1-0.92) = 0.05 \times 0.02 \times 0.08 = 0.00008 \]

\[ P(\text{system works}) = 1 - 0.00008 = 0.99992 \]

Conditional Probability in Engineering

Conditional Probability Formula:

\[ P(A|B) = \frac{P(A \cap B)}{P(B)} \]

Quality Control Application

A manufacturing process has:

• Probability of defect: P(D) = 0.03

• Probability inspection detects defect: P(I|D) = 0.98

• Probability false alarm: P(I|not D) = 0.05

Question: What's the probability a component is actually defective given it failed inspection?

Solution using Bayes' Theorem:

\[ P(D|I) = \frac{P(I|D)P(D)}{P(I|D)P(D) + P(I|\text{not }D)P(\text{not }D)} \]

\[ P(D|I) = \frac{0.98 \times 0.03}{0.98 \times 0.03 + 0.05 \times 0.97} = \frac{0.0294}{0.0294 + 0.0485} = \frac{0.0294}{0.0779} \approx 0.377 \]

Only 37.7% of components that fail inspection are actually defective.

Appendix 1: Statistical Formulas Reference

Quick Reference Guide

Essential formulas and relationships for statistical analysis in engineering applications.

Probability Formulas

Complement Rule: \( P(A') = 1 - P(A) \)

Addition Rule: \( P(A \cup B) = P(A) + P(B) - P(A \cap B) \)

Multiplication Rule: \( P(A \cap B) = P(A|B)P(B) = P(B|A)P(A) \)

Bayes' Theorem: \( P(A|B) = \frac{P(B|A)P(A)}{P(B)} \)

Total Probability: \( P(B) = \sum P(B|A_i)P(A_i) \)

Distribution Relationships

PDF to CDF: \( F(x) = \int_{-\infty}^x f(t) \, dt \)

CDF to PDF: \( f(x) = \frac{d}{dx} F(x) \)

Expected Value: \( E[X] = \int_{-\infty}^{\infty} x f(x) \, dx \)

Variance: \( Var(X) = E[X^2] - (E[X])^2 \)
